Getting Started

Welcome to LLM Agent Playground

This platform provides a unified interface to experiment with multiple large language models including OpenAI, Anthropic's Claude, and Google's Gemini.

Quick Setup

  1. Navigate to Settings and enter your API keys
  2. Visit the Playground to start experimenting
  3. Adjust model parameters using the advanced options
  4. Save your successful prompts in the history section

Tip

Start with simple prompts and gradually increase complexity as you understand each model's behavior.

Playground Features

Core Functionality

  • Multi-Model Support: Switch between OpenAI, Claude, and Gemini with a single dropdown
  • Real-Time Output: See streaming responses as they're generated
  • Prompt History: All your experiments are saved for future reference
  • Model Comparison: Run the same prompt against multiple models side-by-side

Advanced Controls

Temperature

Controls randomness. Lower values make responses more deterministic.

Max Tokens

Limits response length. Higher values allow longer responses.

System Prompts

Predefined roles that guide model behavior.

Dark Mode

Toggle between light and dark themes for comfortable use.

API Integration

Connecting Your API Keys

To use this platform, you'll need API keys from the respective providers:

  • OpenAI API Keys
  • Anthropic API Keys
  • Google API Keys

Sample API Request

Here's how the playground makes requests to the OpenAI API:

fetch('https://api.openai.com/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${apiKey}`
  },
  body: JSON.stringify({
    model: 'gpt-4',
    messages: [{ role: 'user', content: prompt }],
    temperature: temperature,
    max_tokens: maxTokens
  })
})

Security Note

Your API keys are stored locally in your browser and never sent to our servers. However, be cautious about sharing your API keys with others.

Prompt Engineering

Best Practices

Do

  • Be specific about your requirements
  • Provide examples when possible
  • Break complex tasks into steps
  • Specify the desired output format

Don't

  • Use vague or ambiguous language
  • Overload with multiple unrelated requests
  • Assume the model knows your context
  • Forget to set constraints

Prompt Examples

Basic

"Explain quantum computing in simple terms that a high school student would understand."

Intermediate

"Write a Python function that takes a list of numbers and returns the sum of all even numbers. Include type hints and a docstring."

Advanced

"Act as a senior software engineer reviewing this code. Identify potential bugs, suggest improvements, and estimate the time needed to implement them. [paste code]"

Troubleshooting

Common Issues

No response from the model
  • Check your internet connection
  • Verify your API key is valid and has sufficient credits
  • Ensure you've selected the correct model
  • Try reducing the max tokens parameter
Responses are cut off mid-sentence
  • Increase the max tokens parameter
  • Break your prompt into smaller chunks
  • Ask the model to continue where it left off
Model is not following instructions
  • Make your instructions more explicit
  • Provide examples of desired output
  • Lower the temperature parameter
  • Try a different model
